# Docker Utils

## Clean containers
```bash
$ sudo ./clean_containers.sh
```

## Clean images
```bash
$ sudo ./clean_images.sh
```

## Clean volumes
```bash
$ sudo ./clean_volumes.sh
```

## Clean networks
```bash
$ sudo ./clean_networks.sh
```

## Clean all (images, containers, volumes, networks)
```bash
$ sudo ./clean_all.sh
```

## Push docker image (docker hub)
```bash
$ sudo ./push_docker_image.sh
```
