# Docker Utils

## Clean containers
```bash
$ ./clean_containers.sh
```

## Clean images
```bash
$ ./clean_images.sh
```

## Clean volumes
```bash
$ ./clean_volumes.sh
```

## Clean networks
```bash
$ ./clean_networks.sh
```

## Clean all (images, containers, volumes, networks)
```bash
$ ./clean_all.sh
```

## Push docker image (docker hub)
```bash
$ ./push_docker_image.sh
```
